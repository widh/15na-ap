# POSCA Server

This is server which does real-time situation classification.

## Requirements

-   Python 3.6+
    -   `keras` or `keras-gpu`
    -   `websockets`

## How To Run

1. `git clone https://github.com/widh/posca-server`
2. `cd posca-server`
3. Modify `Config.py` properly to meet your condition and models.
4. `python Server.py`
